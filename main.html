Niveles de madurez evaluados

Los niveles de madurez para la evaluación del Sistema de Control Interno están definidos en el marco de referencia Capability Maturity Model Integration (CMMI), y se analizan por cada componente de control. A continuación se describen las definiciones y atributos generales de cada nivel.

5. Optimizado
Los componentes del elemento evaluado cuentan con esquemas de sostenibilidad.
4. Administrativo
Los componentes del elemento evaluado cuentan con esquemas de monitoreo para determinar su gestión.
3. Definido
componentes del elemento evaluado se encuentran documentados, formalizados, divulgados y operando.
2. Repetible
Los componentes del elemento evaluado no cuentan con todas las variables establecidas (documentado, formalizado, divulgado y operando).
1. Inicial
Existen iniciativas al interior de la entidad para desarrollar los componentes del elemento evaluado.
0. No existe
Los elementos evaluados no han sido considerados como relevantes dentro del sistema de control interno de la entidad.
